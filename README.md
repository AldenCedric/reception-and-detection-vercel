# Express.js on Vercel
Basic Express.js + Vercel example that serves html content, JSON data and simulates an api route.

## Clone and Deploy
```bash
npm install
```

```bash
npm run dev
```

```browser
http://localhost:3000
```

## 1. Install Vercel CLI (if not already installed):
```bash
npm install -g vercel
```

## 2. Login to Vercel:
```bash
npm install -g vercel
```

## 3. Deploy:
```bash
npm run deploy
```
